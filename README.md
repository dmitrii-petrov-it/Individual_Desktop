# Individual_Desktop
Приложение TaskManager — это веб-платформа для управления задачами и проектами внутри небольшой или средней команды.
Цель приложения — помочь менеджерам и сотрудникам легко создавать задачи, назначать исполнителей, отслеживать прогресс, оставлять комментарии, хранить вложения и завершать работу по задачам.
Основные функции приложения:
•	управление задачами (создание, изменение, удаление/архивация);
•	назначение задач сотрудникам;
•	контроль статусов;
•	панель (доска задач) с удобной визуализацией;
•	система ролей (Manager / Employee);
•	комментарии и вложения;
•	уведомления и история действий;
•	безопасная аутентификация;
•	API для интеграций.
